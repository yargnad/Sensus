Philosophical Foundation

The Problem: Three Forms of Alienation

1. Social Alienation (The Performance Prison)

Mainstream social platforms have evolved into engines of engagement, trapping users in performative behavior. They reward outrage, gamify validation, and monetize attention at the expense of authenticity.

2. Cognitive Alienation (Outsourced Thinking)

We've delegated our thinking to algorithms, influencers, and echo chambers. We don't know what we believe or why. The capacity for independent thought has atrophied.

3. Civic Alienation (Democratic Collapse)

We lack tools and practices for genuine democratic deliberation. Civic discourse has collapsed into tribalism, misinformation, and power games. Good-faith actors are at a disadvantage.

The Solution: Team Hybrid Philosophy

Human creativity is the fuel. Ethical AI is the engine.

This framework demonstrates that AI doesn't have to optimize for engagement or extraction. It can be affinitiveâ€”creating resonance, facilitating connection, and empowering deliberationâ€”while remaining transparent, accountable, and open-source.

Design Principles

  • Ephemerality over accumulation - Nothing is hoarded, collected, or turned into permanent records
  • Anonymity over performance - No profiles, followers, or metrics; impossible to perform for an audience
  • Deliberation over velocity - Slow down, think deeply, resist the urgency of the attention economy
  • Transparency over opacity - Open-source everything; auditable reasoning; no hidden objectives
  • Local-first over cloud-dependent - Privacy by architecture, not policy
  • Empowerment over automation - AI facilitates human judgment, never replaces it

Act I: Sensus

"Detoxification - Learning to feel again"

Status: LIVE (Deployed)

URL: https://sensus-app-8db18.web.app

Repository: github.com/yargnad/sensus-app

What It Does

Sensus is an anti-social social networkâ€”a one-to-one, anonymous, ephemeral emotional exchange platform where:

  1. Users submit creative expressions (poems, photos, audio) once every 24 hours
  2. Gemini AI analyzes the emotional content to extract 5-10 emotional keywords
  3. The system atomically pairs users with another anonymous submission that shares emotional resonance
  4. Matches are shown once and disappear after 24 hours
  5. Everything auto-deletes after 60 days via MongoDB TTL indexes

Key Features

  • Complete anonymity - No profiles, usernames, follower counts, or public metrics
  • 24-hour cooldown - Prevents addiction loops and gaming the system
  • Atomic matching - Race-condition-free pairing using findOneAndUpdate
  • Graceful degradation - Exponential backoff if Gemini is overloaded
  • Privacy by physics - TTL indexes ensure data can't be leaked because it doesn't exist

Act III: The Lyceum Network

"Infrastructure as Liberation - Building the commons we deserve"

Status: ACTIVE DEVELOPMENT

Target: User-owned, multi-layered decentralized communication and compute network

Repository: github.com/yargnad/The-Lyceum

What It Is

The Lyceum Network is a physical infrastructure projectâ€”a decentralized mesh network that provides communication and compute services as a public commons. It's not owned by corporations or governments; it's owned by the people who run the nodes.

This is the "third act" evolution: from individual detox (Sensus) to personal growth (Whetstone) to collective infrastructure that makes those first two acts sustainable and ungovernable.

The Urgency: Infrastructure Is Power

Every app, service, and digital interaction we have depends on infrastructure controlled by corporations or nation-states. This creates fundamental vulnerabilities:

  • Surveillance - Your data flows through centralized chokepoints
  • Censorship - Platforms can deplatform you at will
  • Extraction - Your attention, data, and creativity are monetized
  • Fragility - Single points of failure, targeted attacks, grid dependence

The Lyceum Network is designed to be resilient, private, and ungovernable by any single entity.

The Three-Layer Architecture

Layer 1: Access Mesh (LoRa)

  • Technology: Long-range, low-power LoRa radio (433-915 MHz)
  • Range: 5-15 km in urban areas, 50+ km in rural/line-of-sight
  • Use Case: Text messages, GPS coordinates, sensor dataâ€”the base layer everyone can access
  • Philosophy: Resilient, works without internet, battery lasts weeks

Layer 2: Utility Fabric (Wi-Fi HaLow)

  • Technology: Sub-1 GHz Wi-Fi (900 MHz HaLow standard)
  • Range: 1-2 km with better penetration than standard Wi-Fi
  • Use Case: Local apps, file sharing, federated AI compute ("Pneuma")
  • Philosophy: Higher bandwidth for local services while maintaining privacy

Layer 3: Backbone Links (Two-Path Model)

  • 900 MHz LoRa PTP: Resilient, low-bandwidth links without line-of-sight (NLOS)
  • 5 GHz Wi-Fi PTP: Fiber-speed bandwidth with clear line-of-sight (LOS)
  • Use Case: Connect regional clusters, create long-haul encrypted backbones
  • Philosophy: Operators choose the best path for their terrain

Key Innovations (Protected as Prior Art)

1. The AI Walkie-Talkie (AIWT)

Voice communication over low-bandwidth mesh:

  • On-device Speech-to-Text (STT) using NPU-accelerated SenseVoice-RKNN
  • Transmit text over LoRa (tiny data footprint)
  • On-device Text-to-Speech (TTS) using paroli library
  • Result: Voice calls over a network that can't normally carry voice

2. "Two-Brain" Power Architecture

  • Listener (SX1262 LoRa): Low-power co-processor stays awake 24/7
  • Thinker (Radxa CPU): High-power main processor sleeps until needed
  • Hardware interrupt wakes the Thinker only when messages arrive
  • Result: Nodes can run for weeks on battery power

3. Pneuma: Federated Compute Network

  • Uses idle NPU/GPU/CPU resources from network nodes
  • "Proof-of-Utility" not Proof-of-Work: Rewards verifiable useful computation, not wasted hash cycles
  • Anti-Blockchain: Lightweight ledger, no mining, no speculation
  • Guardian nodes vet applications for security and performance

4. Symbolon: Vetted Application Ecosystem

  • Containerized applications that run on network nodes
  • Vetted by distributed Guardian nodes (not a centralized app store)
  • Open-source, auditable, and forkable

Design Philosophy: Radical Resourcefulness

We reject engineered obsolescence. The Lyceum honors and utilizes any hardware you can contribute:

  • Brand-new Radxa Rock 5B+ (24GB RAM, NPU for AI)
  • Old Raspberry Pi in a closet
  • Dusty laptop you were going to throw away
  • Android phone with a cracked screen

In the commons, nothing is wasted.

Protecting the Commons: GPL-3.0 and Prior Art

The Lyceum Network documentation establishes prior art for all key architectural innovations, permanently placing them in the public domain to defend against corporate capture.

Every component is GPL-3.0 licensed, ensuring that any derivative work must also remain free and open-source.

    Key Innovations

    Atomic Matching (Sensus)

    Using findOneAndUpdate in MongoDB to prevent race conditions when multiple users submit simultaneously. The match operation is atomicâ€”either it finds a match and updates both records, or it doesn't find a match and saves the submission as "unmatched".

    Philosophical Immune System (All Projects)

    A set of ethical guardrails baked into the system prompt that challenges reasoning, identifies biases, and ensures proposals are tested against multiple ethical frameworks before advancing.

    Local-First AI (The Whetstone)

    Running quantized LLMs on edge devices (OrangePi 5 Plus) ensures conversations never leave the user's home. Privacy by architecture, not policy.

    Decentralized Infrastructure (The Lyceum Network)

    User-owned mesh network using unlicensed radio bands (LoRa, Wi-Fi HaLow, 5 GHz PTP). No ISPs, no corporate surveillance, no single point of failure. Infrastructure as a public commons, protected by GPL-3.0 and prior art.
